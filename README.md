# RL_gridworld
My first little trial on reinforcement learning based on the environment of gridworld. In this repo, I conducted several algorithms of reinfrocement learning on the gridworld environment, including Q-learning, SARSA and torch. I generated trajectory of the agent finding the target point before and after RL.

In the src file folder, I put the base parameters of the research, including arguments, environment of gridworld and the map of training methods.

In the scripts file folder, I put the main file of the training process with several different training methods. It contains random-search-method, q_learning method and SARSA method.
